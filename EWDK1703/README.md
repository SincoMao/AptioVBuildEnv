# AptioVBuildEnv
#### AMI AptioV BIOS Build Environment Integration Package.
<br />从MicroSoft官网（点击底部"*Accept license terms*"）下载[EWDK for Windows 10, version 1703 with Visual C++ Build Tools 2015](https://learn.microsoft.com/en-us/legal/windows/hardware/enterprise-wdk-license-2015)，<br />

解压后将文件夹重命名为 **EWDK1703**，并移动到例如 ***D:\VisualeBios\AptioVBuildEnv*** 目录下。
