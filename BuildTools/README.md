# AptioVBuildEnv
#### AMI AptioV BIOS Build Environment Integration Package.
<br />AMI BuildTools受商业许可限制，需要授权账号从[Customer Portal](https://cp.ami.com)下载。<br />

将所需版本的"**Aptio_5.x_TOOLS_xx**"下载到本地，解压后仅将"**BuildTools**"文件夹复制出来归纳好并依据版本重命名， 

例如将"**Aptio_5.x_TOOLS_JRE_37_1.zip**"解压后只需把里面的"**BuildTools**"文件夹重命名为"**37.1**"，  

然后移动到例如 ***D:\VisualeBios\AptioVBuildEnv\BuildTools*** 目录下，另一个文件夹"VisualeBios"为IDE工具可以不用。
